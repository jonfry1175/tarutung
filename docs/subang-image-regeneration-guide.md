# Panduan Regenerasi Image Subang 360

Dokumen ini mencatat **19 file image Subang yang aktif digunakan website** pada 21 Juli 2026. Setiap file ditampilkan bersama label, posisi pemakaian, referensi foto nyata yang ditemukan melalui Google Images, dan prompt siap pakai untuk image-gen.

## Cara menggunakan referensi

1. Unggah **Image 1** dari tautan referensi sebagai acuan fakta visual: bentuk lokasi, vegetasi, warna, material, makanan, pakaian, dan suasana nyata.
2. Unggah **Image 2** dari aset website saat ini sebagai acuan komposisi dan rasio saja.
3. Jalankan prompt pada bagian file terkait. Jangan meminta model menyalin orang, logo, tulisan, watermark, atau detail berhak cipta dari foto referensi.
4. Kecuali referensi Wikimedia Commons yang lisensinya disebutkan, foto sumber di bawah adalah **referensi visual saja**. Jangan menerbitkan ulang foto aslinya tanpa izin pemilik.
5. Setelah regenerasi, periksa lagi anatomi manusia, jumlah jari, bentuk alat makan, tulisan, geografi, spesies tanaman, dan detail budaya.

Aturan realisme yang berlaku untuk semua prompt: dokumenter perjalanan/editorial, cahaya alami, dynamic range masuk akal, tekstur tidak sempurna, sedikit grain kamera, warna Indonesia yang natural, tanpa HDR berlebihan, tanpa glow fantasi, tanpa kulit plastik, tanpa simetri artifisial, tanpa teks, tanpa logo, dan tanpa watermark.
